# Images for IELTS Speaking Test

This directory should contain images used in Part 1.2 (Picture Description) of the IELTS Speaking Test.

## Required Images:

1. **family_park.jpg** - Family scene in a park with people doing activities
2. **office_meeting.jpg** - Business meeting or office environment  
3. **restaurant_scene.jpg** - People dining in a restaurant

## Image Requirements:

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 800x600 pixels or similar aspect ratio
- **Quality**: Clear, high-resolution images suitable for description
- **Content**: Appropriate for language learners, showing clear activities and relationships

## Adding New Images:

1. Place image files in this directory
2. Update the corresponding questions in `data/questions.js`
3. Ensure the filename matches the "image" field in the question data

## Sample Images Sources:

For testing purposes, you can download appropriate images from:
